优势分析

1 灵感到发布的完整管道（独创）

  • 5 阶段生命周期比 Trello/Todoist 的简单状态更贴合独立开发者的实际工作流
  • "Ideas Pool + Task Board" 双页面分离了发散思维和执行跟踪
  • "publish" 状态超越了普通的 "done"，对需要发布/上线的一人公司来说很实用

2 零成本自托管，完整数据所有权

  • JSON 文件存储，git-friendly，可以直接版本控制
  • 不依赖任何第三方 SaaS，没有订阅费用和数据锁定
  • 对比 Linear ($8/月)、Notion (团队付费)，长期成本为零

3 极简架构，极低运维负担

  • 全栈 ~1000 行代码，没有数据库、没有构建工具、没有 Docker
  • npm start 一键启动，对比 Jira 的部署复杂度几乎可忽略
  • 不需要学习框架，任何会 HTML/JS 的人都能改

4 离线韧性

  • API 不通时自动回退 localStorage，不会中断工作
  • 对比 Trello 离线时基本不可用，这是实际的生产力保障

5 语音速记灵感

  • Web Speech API 直接录入，不需要切换 app
  • 对比 Todoist/Things 需要手动输入，语音捕捉灵感的摩擦力更低

6 拖拽 + 标记的轻量交互

  • Kanban 拖拽跨列 + 标记循环交互密度适中
  • 对比 Linear 的复杂工作流、Jira 的配置地狱，更适合一个人决策
